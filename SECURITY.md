# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability, please send an email to contact@peviitor.ro

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Any suggested fixes (optional)

We will:
1. Acknowledge your report within 24 hours
2. Provide a timeline for fix
3. Credit you in the security advisory (if desired)

## Security Best Practices

When using this project:

- Never commit credentials or API keys to the repository
- Use environment variables for sensitive data
- Keep your Chrome and Docker installations updated
- Review job data before pushing to production Solr
